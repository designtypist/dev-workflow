# Dev Workflow

> Objective: Documentation of my development workflow with an aim to improve my Action per Minute.

---

## Quickstart Guide

#### 1. Setup
```
npm i docsify-cli -g
git clone git@github.com:designtypist/dev-workflow.git
cd dev-workflow/
```

#### 2. Host Locally
```
docsify serve docs
```
Navigate to `http://localhost:3000`
